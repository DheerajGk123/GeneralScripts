Installing Ubuntu Alongside Windows 10


Backup your hardisk to regain the data again in case of any hardisk failure


1. Go to BIOS setup by pressing ESC as soon as system begins . If not working refer your computer manufacturer on how to enter BIOS




2. Go to System Configuration




3. In Legacy Boot Order , put USB Diskette on Key/USB Hard Drive on top of the list







4. Press F10 to save and exit


5. You should see the following image




6. Now click on Install Ubuntu




7. Select your language and test your keyboard by typing something in the field provided and click Continue




8. If necessary connect to a network ( not mandatory )




9. If you want to connect , enter the network password




10. Click Continue




11. Select Normal Installation and in Other Options field , check both the boxes and click Continue




12. Select Install Ubuntu Along Windows 10 and click Continue




13. Select a minimum of 10GB . Can be adjusted by dragging the border left/right border of the boxes




14. An alert pops-up . Click Continue




15. Another alert pops-up . Click Continue


16. Select your Timezone




17. Set your Personal PC information and click on Continue




18. The installation begins
