Safe Removing of Ubuntu Installation (Uninstall)


Backup your hardisk to regain the data again in case of any hardisk failure


If you want to remove the entire Ubuntu Operating System in a clean way follow these procedures (windows 10 or any other OS will remain UNAFFECTED



Resources Required

1.     A Boot-Repair-Disk

2.    Unet Booting Software

3.    Download Ubuntu for Desktop




Create a Boot Repair Disk Using Unet Booting


1. Insert the Flash Drive


2. After downloading UnetBooting , open it






3. Check the Disk Image Option and click on Select




4. Select the Ubuntu iso file downloaded




5. Make sure your flash drive is selected in the Drive field






6. Now click on Ok


7. Wait for the process to complete







Procedure

1. Insert the flash Drive into your computer


2. Then start the system via bootable disk just created (pendrive)


3. the following image should appear shortly






4. In the Boot Repair interface , select Advanced Options




5. In Main Options Tab , check Restore MBR and select Apply




6. Wait for the process to complete




7. When Completed , shut down (power button can be found at bottom-right)




8. Remove the Flash Drive and start the computer


9. You should boot into Windows normally
