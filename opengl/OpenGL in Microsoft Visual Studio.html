Requirements


  1. Microsoft Visual Studio
  2. OpenGL





Installation


  1.   Paste glut.dll in Local Disk C -> Windows -> System32

  2.   Paste glut32.dll in Local Disk C -> Windows -> System32

  3.   Paste glut.h in Microsoft Visual Studio -> VC -> Tools-> MSVC -> 14.24.28314 -> include -> GL

  4.   If GL folder is not present, create a folder and name it GL

  5.   Paste glut.lib in Microsoft Visual Studio -> VC -> Tools -> MSVC-> 14.24.28314 -> lib -> x64

  6.   Paste glut32.lib in Microsoft Visual Studio -> VC -> Tools -> MSVC-> 14.24.28314 -> lib -> x86





Setting up OpenGL in IDE


  1.   Open Microsoft Visual Studio

    2. cant find it, try this
    Microsoft Visual Studio -> Common7 -> IDE -> devenv.exe


  2.  Create a New Project


  3.  Console App (click next)


  4.  Enter a project name Ex. demo and select the location to save the project (click create)


  5.  On the right side, right click on demo


  6.  Select properties (Property Tab appears)

  7.  In Configuration, select All Configuration

  8.  Under Configuration Properties,
    C/C++ -> Precompiled Headers -> (click) Precomplied Header

    9. change the value of Precompiled Header to Not using precompiled header


  9.  Now back to Configuration Properties,
    Linker -> Input -> Additional Dependencies -> (right click) -> edit

    10. Add these 3 lines :

    opengl32.lib
    glu32.lib
    glut32.lib


  10.  Close The Tab
